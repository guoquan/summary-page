== Description ==

List and/or include sub-pages with meta option.

After install and activate the plugin, add a custom field 'is_summary' and set to 1 to enable for each individual page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `summary-page.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a custom field 'is_summary' and set to 1 to enable for each individual page.
